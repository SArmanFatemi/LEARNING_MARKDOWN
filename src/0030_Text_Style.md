
# Simple text styles
Text can be styled as italic,  bold or strikethrough.

## Italic
* *First Method*
* _Second Method_

## Bold
* **First Method**
* __Second Method__

## Strikethrough
* ~~Some Sample Text~~

## The Combination
* ~~***First Method***~~
* ~~**_Second Method_**~~
* ~~*__Third Method__*~~
