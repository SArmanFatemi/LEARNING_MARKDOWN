# Headings
You can create HTML Headings by prepending the text you want to be in that element by a number of (#).

# Heading 1  - `<h1>` Tag
## Heading 2  - `<h2>` Tag
### Heading 3  - `<h3>` Tag
#### Heading 4  - `<h4>` Tag
##### Heading 5  - `<h5>` Tag
###### Heading 6  - `<h6>` Tag
